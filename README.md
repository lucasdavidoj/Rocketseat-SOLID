# Rocketseat-SOLID

Esta é uma API Rest desenvolvida como desafio ao longo do bootcamp de NodeJS da Rocketseat.

## :screwdriver: Setup

```bash
yarn
# ou
npm install
```

## :rocket: Run

```bash
yarn dev
# ou
npm run dev

# A aplicação iniciará em:
http://localhost:3333/

# A documentação poderá ser acessada em:
http://localhost:3333/api-docs/
```
